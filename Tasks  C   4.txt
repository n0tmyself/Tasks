Задание 7.a (строки)

int n = 2;
long l = 33;
float f = 3.1415926;
double d = 2.718281828;
char s[1024];

С помощью функций sprintf и snprintf заполнить массив char значениями переменных int, long, float, double следующего формата:
"variable n = 2"
"variable l = 033 = 21h" (вывести 3 как трехзначное десятичное число и как шестнадцетиричное %.3d %X)
"variable f = 3.14" (с точностью до второго знака %.2f)
"variable d = 002.71828" (как трехзначное число целой части плюс дробная часть с точностью до 5 знака)

Узнать все особенности форматирования %... чисел семейства функций printf, sprintf, snprintf.


Задание 7.b

Каждое предложение разбить на слова (слова должны хранится в std::list<std::string>).
Определить какие буквы использованы в тексте и вычислить, сколько раз использовалась каждая буква (с помощью std::map).
